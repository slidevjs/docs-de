## Server lokal starten

```
npm i -g pnpm

pnpm i
pnpm run dev
```

Und besuche `http://localhost:3000`

Oder installiere die [Vite extension for VS Code](https://marketplace.visualstudio.com/items?itemName=antfu.vite) um Seite an Seite zu editieren.
